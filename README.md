# Senfo <sub>lang</sub> v2

## What we have
- Numbers - 0-9
- Hex numbers - (ex.: #2def)
- Float numbers - (ex.: 3.14)
- **String**
- Operators
  - unary
    - print - operator for expression log
  - binary - (-, +, /, *, ||, &&, ==, !=, >, >=, <, <=)
- Brackets - ()
- Variables
- Constants (PI, E, GOLDEN_RATION)
- Comments
  - One line - // comment
  - Multiline - /* comment */
- if, for, while, do/while
  - continue, break
